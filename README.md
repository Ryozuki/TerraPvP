## TerraPvP

Author: Ryozuki

TerraPvP is a plugin for TShock that adds a pvp ranking ladder with a qeue.

## Commands:

- /tjoin
- /tleave
- /tstats
- /tlist
- /tlistp
- /tcreate
- /taddspawn
- /tsave
- /tdel
- /ttopten

## Permissions
- terrapvp.join (/tjoin, /tleave, /tlist, /tlistp)
- terrapvp.stats (/tstats, /ttopten)

### Admin
- terrapvp.arena (/tsave, /tcreate, /tsetspawn, /tdel)
